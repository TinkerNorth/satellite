How to install ViGEmBus

  1. Download the latest ViGEmBus release from github.com/nefarius/ViGEmBus
  2. Run the installer (ViGEmBus_Setup_x64.msi)
  3. Restart may be required — check Device Manager under System devices for "Nefarius Virtual Gamepad Emulation Bus"
  4. Come back here and the status should show Detected

ViGEmBus is a kernel driver that lets Satellite create a virtual Xbox 360 controller on your system. Without it, controller input cannot be forwarded.
