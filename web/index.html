    What your donation actually pays for

    • Hosting. dish.tinkernorth.com, tinkernorth.com, and every signed-installer mirror.
    • Code-signing certificates. Windows SmartScreen only plays nice because we pay for an EV certificate every year, and the same goes for the Apple Developer Program account for the upcoming macOS client.
    • Google Play developer fee. Plus the time required to keep the Play listing compliant with every new policy round.
    • Hardware testing. Phones across the Android version range, gaming PCs at different tiers, and the controllers we benchmark against.
    • Time. Honest answer: most of it. This is a nights-and-weekends project. Donations let me say "yes" to working on it instead of taking on extra consulting.

    Donation FAQ

    Where does the money actually go?

    Code-signing certificates, the Google Play developer fee, hosting, the eventual Apple Developer Program account for Dish for macOS, domain renewals, and the time I'd otherwise spend taking contract work instead of shipping.

    Is TinkerNorth a registered non-profit?

    No. TinkerNorth is an independent maker, so donations are not tax-deductible. They're treated as gifts.

    Which platform should I use?

    GitHub Sponsors if you can — it takes 0% platform fees on individual sponsorships and makes recurring support trivial. Ko-fi is great for one-off tips. Buy Me a Coffee works well if you'd rather pay with Apple/Google Pay and don't have a GitHub account.

    Do sponsors get anything special?

    Nothing functional. Every feature, every release, every privacy guarantee is the same for sponsors and non-sponsors. What you get is a shout-out in the release notes for major versions and the knowledge that you're directly funding a free-as-in-freedom project.

    I can't donate. How else can I help?

    Star the repos on GitHub, file good bug reports, contribute a PR, write a blog post or video, or just tell a friend who's been hunting for a wireless gamepad solution.

    Can't donate? You can still help.

    • Star the Satellite repo and the Dish for Android repo on GitHub. It genuinely moves the needle for discovery.
    • File good bug reports. A reproducible issue is worth ten "doesn't work" threads.
    • Tell a friend. A single post or short video is worth a lot.
    • Contribute. Pull requests are welcome on every repo — see CONTRIBUTING.md. Translations are particularly easy to land.

    Thank you. Every dollar, every star, every shared link. They all add up. ~Emir
